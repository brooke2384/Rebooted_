# **Rebooted - Sustainable Tech eCommerce & Repair Platform**

### **A Next.js eCommerce platform for refurbished tech products and repair services.**

---

## **Overview**

Rebooted is an eCommerce site focusing on sustainable tech. We offer refurbished products, repair services, and solutions that extend the lifespan of technology. Built with Next.js for performance and SEO, it’s mobile-responsive, user-friendly, and designed to promote eco-friendly tech consumption.

---

## **Features**

- **Home**: Introduces Rebooted’s mission and highlights featured products.
- **About Us**: Our sustainability goals and team.
- **Products/Services**: Product listings with filtering and search.
- **Solutions**: Business solutions and case studies.
- **Resources**: Guides and articles on sustainable tech.
- **Community**: User discussions and event highlights.
- **Contact Us**: Inquiry form and company info.
- **Account**: Manage orders, preferences, and reviews.
- **Cart & Checkout**: Integrated shopping cart and secure payments with Stripe.

---

## **Technologies**

- **Frontend**: Next.js, React, TailwindCSS
- **Backend**: Node.js, Express (optional)
- **Authentication**: JWT
- **Payments**: Stripe/PayPal
- **Deployment**: Vercel

---

## **Installation**

1. Clone the repo:  
   `git clone https://github.com/yourusername/rebooted.git`
2. Install dependencies:  
   `yarn install`
3. Create a `.env.local` file with environment variables.
4. Run the development server:  
   `yarn dev`
